# ☕ Premium Coffee Shop

Premium Coffee Shop is a modern and fully responsive landing page designed to showcase premium coffee products and services. The website focuses on clean design, accessibility, and a smooth user experience across all devices.

## Features

- Fully Responsive Design
- Mobile-First Approach
- Sticky Navigation Bar
- Smooth Scrolling Navigation
- Mobile Hamburger Menu
- Contact Form with Client-Side Validation
- Accessibility Support (ARIA attributes, keyboard navigation, alt text)
- Semantic HTML Structure
- Clean and Maintainable Code

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Flexbox
- CSS Grid

## Getting Started

1. Clone or download the repository.
2. Open the `index.html` file in your browser.
